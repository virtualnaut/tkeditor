About tkEditor

What is tkEditor?

TkEditor is a tool for arranging widgets in a user interface. Designed user interfaces can then be exported to Python so users can add functionality using the IDE of their choice.

Tools Used to Create/Manage tkEditor

  • Notepad++
  • Wing Python IDE
  • GIMP
  • Git
  • TortiseGit
