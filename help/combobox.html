tkEditor Help

ttk.Combobox

Description

The ttk.Combobox widget (also called a checkbox) is a widget that allows the user to select an item from a list of options.

Specific Properties

Properties Description Data Type
values The list of values that the user can pick from. List
takefocus Whether the widget can be selected using the TAB key. Boolean
cursor The cursor that is displayed then the widget is moused over. String
