tkEditor Help

ttk.Progressbar

Description

The ttk.Progressbar widget graphically displays a percentage.

Specific Properties

Properties Description Data Type
length The length. Measured in number of pixels. Integer
orient Whether the progress bar is displayed vertically or horizontally. String
maximum The maximum value of the progress bar. Integer
mode Whether the progress bar displays a percentage, or a generic loading animation. String
cursor The cursor that is displayed then the widget is moused over. String
