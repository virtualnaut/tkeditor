tkEditor Help

Widgets

What is a Widget?

A widget is a part of a GUI (graphical user interface), that can display information or be interacted with, like a button or text label, for example.

What is a Property?

In tkEditor, widgets have a number of different properties that can be changed to change how a widget appears or works.

General Properties

All tkEditor widget have these properties in common:

Properties Description Data Type
Identifier The variable name that is used to access the widget in the python code. String
X-Coord The position of the widget on the x-axis.* Integer
Y-Coord The position of the widget on the y-axis.* Integer

*The coordinate origin (0,0) is at the top left corner of the window.

Supported Widgets

  • ttk.Button
  • ttk.Checkbutton
  • ttk.Combobox
