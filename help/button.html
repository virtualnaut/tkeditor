tkEditor Help

ttk.Button

Description

The ttk.Button widget is a widget that can be pressed by the user, and can be programmed to call a subroutine when a press event happens.

Specific Properties

Properties Description Data Type
width The width of the button. Measured in number of characters.* Integer
text The text displayed on the button. String
takefocus Whether the widget can be selected using the TAB key. Boolean
cursor The cursor that is displayed then the widget is moused over. String

*To convert this to pixels, multiply the number of characters by the average width of a character in the font being used.
