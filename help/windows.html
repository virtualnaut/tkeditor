tkEditor Help

Windows

Menu Window

The menu window is used for performing various project operations:

  • New - start a blank project.
  • Load - open an existing .tkw project file.
  • Save - save the current project to a .tkw file.
  • Export - export the current project to a Python file.
  • Help - open the help pages.

Display Window

In a new project, this window is titled 'Untitled'. This window is where widgets appear and can be moved around by clicking and dragging them.

Selection Window

The display window is where the properties of the currently selected widget are displayed and can be edited. To edit a property, double click on it.

Widget Add Window

The widget add window is used to add widgets to the project.
